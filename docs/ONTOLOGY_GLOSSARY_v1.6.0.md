# ONTOLOGY GLOSSARY - EL FOTÓGRAFO v1.6.0

Este glosario define los términos de dominio para asegurar la consistencia entre la UI y el código.

| Término Técnico (Código) | Término de Dominio (UI) | Definición |
| :--- | :--- | :--- |
| **initImageBase64** | **Negativo Digital** | La imagen original cargada por el usuario antes de cualquier proceso. |
| **FacialTraits** | **Firma Facial / ADN** | Conjunto de rasgos biométricos extraídos mediante análisis de IA. |
| **EngineMode: live** | **Sensor Full Frame** | Uso de modelos Pro (Gemini 3 Pro) con validación avanzada. |
| **EngineMode: offline** | **Sensor Standard** | Uso de modelos Flash para velocidad y eficiencia. |
| **Pose** | **Esquema de Composición** | La estructura física y postura del sujeto en la escena. |
| **STYLE_MAP** | **Set de Luces** | Definición de la atmósfera lumínica (Natural, Noir, Neón). |
| **SKIN_MAP** | **Química / Grano** | Texturizado de piel y post-procesado analógico. |
| **RenderItem** | **Revelado** | El resultado final procesado por el laboratorio. |
| **savedRenders** | **Archivo Maestro** | Almacén permanente de negativos finalizados. |
| **renderLibrary** | **Hoja de Contactos** | Historial temporal de la sesión actual. |
| **analyticsService** | **Bitácora de Exposición** | Registro técnico de calidad y rendimiento del sensor. |
| **GearSettings** | **Equipo Óptico** | Selección de lentes (35mm, 50mm, 85mm) y película. |
